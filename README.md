# 博客

> 已经由 Hexo 迁移到 [Astro](https://astro.build/)，主题用的是 [astro-theme-typography](https://github.com/moeyua/astro-theme-typography)。

## 使用

```bash
# 安装依赖
pnpm install
# 启动服务
pnpm run dev
```

## 新建文章

```bash
pnpm run new-post
```

## 更新主题

```bash
pnpm run update-theme
```

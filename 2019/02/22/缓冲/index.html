发表于 2019-02-22 | 分类于 JavaScript

一、缓冲

一个盒子用3000毫秒时间，从100→700，不一定是匀速的。
想象一下小时候升国旗，国歌55秒，旗杆10m。小孩子总能55秒准确的升到10m顶端，过程大家懂得。

过程到底是匀速的、还是先快后慢、先慢后快呢？这就是 缓冲，英语叫做tween。
缓冲的实现，非常简单，就是我们的数学家给我们提供了非常多的缓冲算法，都很好用，一个经典的缓冲。

他们的参数，都是t、b、c、d：
第一个参数t表示当前帧编号
第二个参数b表示起始位置
第三个参数c表示变化量
第四个参数d表示总帧数
函数的返回值，就是t这一帧，元素应该在的位置。
